USEFUL COMMANDS:

Make Directory:
mkdir

Move Directory:
mv source dest
-or-
mv /path/to/source /path/to/dest

Change Directory:
cd

View Directory:
ls

Clear Screen:
clear

Exit Terminal:
exit

Terminate Program in Terminal:
CTRL + Z

Create a new Document:
vim document_name.file_extension

Where am I at/ what directory am I in:
pwd

Copy File Command:
cp file_original.cpp file_new.cpp

USEFUL C++ Commands:
g++ -o compiled_program_name program_literal.cpp // Compile
./compiled_program                               // Run

USEFUL Git Commands:
git add --all
git commit

USEFUL VIM Commands:
"esc" key to exit insert mode
"i" key to insert
:wq to quit with save
